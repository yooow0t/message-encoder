# message-encoder
Presents an array of different rudimentary methods for encoding and decoding a message

# Supported Methods
+ Affine Cipher
+ Keyed Ceasarian
+ Ceasarian Shift
+ ROT13
+ Vigenere Cipher
+ Keyed Vigenere
+ Autokeyed Vigenere
+ Gronsfeld Cipher

As of this moment only encoding is supported. More methods and decoding support coming soon.
